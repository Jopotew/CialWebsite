# 🏢 CIAL Website

Proyecto empresarial desarrollado con **React**, **Vite** y **Tailwind CSS**.

## 🛠️ Instalación y Uso Local

1. Clona el repositorio.
2. Ve a la carpeta del proyecto: `cd files`.
3. Instala dependencias: `npm install`.
4. Corre el desarrollo: `npm run dev`.

## 🚀 Despliegue en GitHub Pages

Este proyecto utiliza `push-dir` para enviar la carpeta `dist` a la rama `gh-pages`.

1. Asegúrate de que tus cambios estén guardados (`git commit`).
2. Ejecuta:
   ```bash
   npm run deploy
